# 004b — dense news matome

## 狙い
ニュース速報系まとめサイトのUI。速報帯・タブ切り替え・時系列タイムラインで情報密度を最大化。ダークヘッダー＋赤アクセントで速報感を演出。

## 強いところ
- 速報帯（BREAKING）でトップニュースが常に目に入り、速報性の高い記事への回遊が増える
- タイムライン形式で時刻・カテゴリ・riskが一目で把握でき、スキャンしやすい
- タブ（新着/速報/人気/カテゴリ別）で同一ページ内のフィルタリングを想定
- カテゴリ別カラーバッジで視認性が高い

## 弱いところ
- 情報密度が高い分、初見ユーザーには圧が強い
- 速報帯のスクロールはJSなしでは静的表示のみ（本番ではCSS animationまたはJS追加が必要）
- ニュース速報メインだとアフィリエイト導線が弱く、収益構造の設計が別途必要

## 本番採用するなら
- **残す**: 速報帯、タイムラインリスト、カテゴリカラーバッジ、PR表記
- **削る**: スレッド風レス表示（ニュース速報には冗長）、タブは本番ではJS実装要
- **追加検討**: WebSocket/SSEで速報リアルタイム更新、Push通知、RSS自動取り込みパイプライン
