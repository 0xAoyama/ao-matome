## Variant: Classic 2ch Matome

### Design stance
伝統的な2ch/掲示板まとめサイトのUIパターンを、matrix-aoの比較用に再構成したモックです。

### Key choices
- 青いヘッダー + 黄色アクセント + 白背景 + 2カラム
- 速報/人気記事/ランキング/広告枠/カテゴリサイドバー
- レス風の記事本文、番号、緑の投稿者メタ、赤い強調
- ただし実運用では掲示板本文・画像・SNS投稿本文は転載しない前提
- 公式リンク、確認日、risk、転載なしラベルを明示

### Strong at
- 伝統的まとめサイトとしての分かりやすさ
- 情報密度、ランキング、回遊導線
- SEO/広告運用っぽい既視感

### Weak at
- 古く見える
- 信頼性・ブランド感は弱い
- 広告過多/低品質まとめサイトの印象を持たれやすい
- matrix-aoの「公式情報・低リスク」方針とは緊張関係がある

### Best use
本番デザインの採用候補というより、回遊導線・ランキング・新着一覧・サイドバー構成の参考として使うのが現実的です。
