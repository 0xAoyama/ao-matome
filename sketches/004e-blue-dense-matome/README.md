# 004e: 青×高密度 汎用トップ候補

## 狙い

004b(ダーク/赤・高密度・速報感)と004d(青ヘッダー/黄アクセント・角丸・余白)を統合した汎用トップページ候補。青基調で速報感と読みやすさを両立させる。

## 004bから継承したもの

- **速報帯(Breaking Ticker):** 画面上部の赤背景スクロールバーで速報性を演出
- **タイムライン密度リスト:** 時刻・カテゴリ・タイトル・リスクバッジを1行に凝縮した高密度リスト
- **タブ切り替え:** 新着/速報/人気の切り替えUI
- **高密度レイアウト:** 1画面に多くの情報を表示する設計思想

## 004dから継承したもの

- **青ヘッダー(#2f5fa7)/黄アクセント(#e7b800):** 落ち着きと視認性を両立するカラー
- **角丸(border-radius: 6px):** カード・バッジに統一的な角丸を適用
- **余白:** 004bより余白を増やし、読みやすさを確保
- **バッジUI:** カテゴリ・リスク・速報のバッジスタイル
- **ピックアップグリッド:** 3カラムのピックアップセクション

## 強いところ

- **3ジャンル横断(AI・テック / ゲーム / 美容・生活)** で汎用性が高い
- 速報帯+タイムラインで**速報性を維持**しつつ、青基調+余白で**読みやすさも確保**
- コンプライアンスUI(転載なし表示、確認日、リスクバッジ、PR表記、YMYL注意)を全て網羅
- モバイルレスポンシブ対応(860px以下で1カラム化)

## 弱いところ

- ジャンル特化デザイン(004a的なゲーム特化、004c的なAI特化)と比べると**個性が薄い**
- 3ジャンル分のコンテンツを1ページに載せるため、**コンテンツ量が増えるとスクロールが長くなる**
- 速報帯の赤と全体の青の配色バランスに注意が必要

## 本番採用するなら

### 残す要素

- 青ヘッダー+黄アクセントのカラースキーム
- 速報帯(ニュース速報がある場合のみ表示に変更)
- タイムライン密度リスト(メインコンテンツとして最重要)
- コンプライアンスUI一式(転載なし、確認日、リスク、PR表記、YMYL方針)
- タブ切り替え
- 2カラムレイアウト+モバイル1カラム

### 削る/調整する要素

- ピックアップグリッドは記事数が少ないうちは不要、コンテンツ量に応じて表示
- サンプル記事+コメント欄はトップページからは除外し、個別記事ページに移動
- サイドバーの広告枠は実際の広告が入るまでは非表示
- ジャンルが増えた場合はトップを「総合」に絞り、ジャンル別ページに分離
