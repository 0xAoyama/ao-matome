# 004f: AI・テック特化ブルーニュース

## 狙い

AI・テック特化の速報まとめサイト。004bの速報タイムラインを最大限活かしつつ、004dの青基調デザインで統一した案。テック系読者が「ここだけ見ればAI業界の動きがわかる」と思えるハブを目指す。

## 004bから継承したもの

- **速報ティッカー帯** — ページ最上部に常時流れる速報。この案では最も強く打ち出している
- **密度の高いタイムライン** — 10件以上のニュースを一覧表示、バッジで分類（LLM/エディタ/API/クラウド）
- **タブ切り替え** — 速報・LLM・エディタ・API/料金・クラウドでフィルタリング
- **スレッド形式コメント** — 2ch/5chライクなレス構造

## 004dから継承したもの

- **青ヘッダー**（#1a3a6b / #2f5fa7のグラデーション）
- **角丸カード**（border-radius: 8px）
- **バッジUI**（ソース・リスク・PR・YMYL）
- **読みやすいフォント設定** — system-ui + Hiragino + Noto Sans JP、line-height: 1.7
- **2カラムレイアウト**（max-width: 1180px）

## 強いところ

- **情報密度がテック系読者に刺さる** — タイムライン+比較表で「このサイトだけ見ればいい」感
- **比較表が差別化要素** — 料金・性能を一覧で比較できるのは他のまとめサイトにない
- **速報ティッカーの存在感** — リアルタイム感があり、リピート訪問を促す
- **コンプライアンスUI完備** — 確認日・ソース明記・YMYL注意・PR表記で信頼性を担保

## 弱いところ

- **ニッチすぎてPV規模が限られる** — AI/テックに興味のない層は来ない
- **美容・ゲーム・芸能読者は完全に対象外** — 汎用まとめサイトとしてのスケールは期待できない
- **料金情報のメンテナンスコスト** — 比較表は常に最新に保つ必要があり、運用負荷が高い
- **速報ティッカーの運用** — 自動化しないと人力更新がボトルネックになる

## 本番採用するなら

### 残す要素

- 速報タイムライン（コア機能）
- 比較表セクション（差別化の柱）
- コンプライアンスUI一式
- 青基調のカラースキーム
- タブによるカテゴリフィルタ

### 削る/改善する要素

- ティッカーはRSS/API連携で自動化必須（手動更新は非現実的）
- 比較表はCMS化してスプレッドシート連携が必要
- コメント欄はMVPでは省略し、外部サービス（Disqus等）で代替も検討
- ランキングはアクセス解析連携で自動生成に
- モバイルでのタイムライン表示をさらに最適化（現状は最低限のレスポンシブ）
