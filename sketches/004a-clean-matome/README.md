# 004a — clean matome

## 狙い
004（古典的2chまとめUI）をベースに、配色を青/白/黄に統一し、バッジ・ランキング帯など信頼感を上げるパーツを追加。情報密度は維持しつつ「清潔・汎用」寄りに調整した案。

## 強いところ
- ランキング帯（横5列）が回遊を促し、PV/滞在を稼ぎやすい
- バッジ（公式ソース / low risk / PR）で記事の性質が一目で分かる
- 004の2カラム+サイドバー構造をそのまま活かしており既存まとめ読者に違和感が少ない
- 青/白/黄の配色が汎用的で、カテゴリを問わず使える

## 弱いところ
- 004とほぼ同じ構造なので差別化が薄い — 004で十分という判断もありうる
- ランキング帯は横5列でモバイルだと崩れやすい（レスポンシブで縦積みにフォールバック済み）
- バッジが増えるとごちゃつくリスク

## 本番採用するなら
- **残す**: ランキング帯、バッジ体系、PR表記、ポリシー表示
- **削る**: スレッド風レス表示は本番では不要かもしれない（公式要約カード＋コメント欄で代替）
- **追加検討**: パンくずリスト、ページネーション、構造化データ
