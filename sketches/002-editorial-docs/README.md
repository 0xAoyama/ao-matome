## Variant: Editorial docs / Mintlify × Notion

### Design stance
Editorial docs / Mintlify × Notion を基調にした matrix-ao のデザイン方向性モックです。

### Key choices
- Layout: トップページのヒーロー、カテゴリ、最新記事/注目記事を一画面で比較できる構成
- Typography: 参照デザインに合わせたGoogle Fonts代替
- Color: 参照デザインの主要トークンをmatrix-ao向けに再解釈
- Interaction: hoverまたはチップ切替/軽いアニメーションを含む

### Trade-offs
- Strong at: 第一印象、方向性判断、実装前の比較
- Weak at: 本番Astroコードではないため、選定後に実装へ移植が必要

### Best for
- Editorial docs / Mintlify × Notion の雰囲気がmatrix-aoに合うかを判断するための初期比較
