# 004c — anime/game matome

## 狙い
アニメ・ゲーム・推し活に特化した古典まとめUI。紫/オレンジのポップ配色で趣味メディア感を出しつつ、発売日/配信日/イベント導線を前面に配置。著作物画像は使わずプレースホルダで権利処理の安全性を担保。

## 強いところ
- イベント・発売日・配信日の帯が目立ち、「次に何がある？」がすぐわかる
- サムネグリッド4列でビジュアル密度が高く、推し活ユーザーの回遊を促進
- カテゴリカラー（アニメ紫/ゲーム緑/イベントオレンジ）で視認性良好
- 著作物画像プレースホルダで権利リスクゼロ

## 弱いところ
- サムネがプレースホルダのみなので本番では公式OGP取得やアフィリエイト画像の利用可否検討が必要
- ポップ配色がニュース系・生活系記事と合わない — ジャンル特化にするなら他カテゴリは別テーマが必要
- 推し活ユーザーは画像がないとエンゲージメントが落ちる可能性

## 本番採用するなら
- **残す**: イベントカレンダー帯、サムネグリッド構造、カテゴリカラー、PR/確認日バッジ
- **削る**: スレッド風レスは推し活メディアでは不要（カード型公式要約で代替）
- **追加検討**: 公式OGP画像の自動取得、配信サービスロゴ表示（利用規約確認要）、推し活カレンダーのiCal出力
