# 004h: コンプレックス/美容特化（青・信頼重視）

## 狙い

コンプレックス/美容ジャンル（メンズ脱毛・体臭ケア・ホワイトニング・スキンケア・AGA）に特化したまとめサイトのスケッチ。全バリアントの中で **YMYL対策を最も強く打ち出した案**。004b（情報密度・速報性）と004d（青基調・信頼感デザイン）のハイブリッド。

## 継承元

### 004b から継承
- タイムライン形式による速報性の表現
- 情報密度の高いレイアウト（2カラム、テーブル、スレッド）
- スレッド形式コメントによる多角的情報提示

### 004d から継承
- 青基調（#2f5fa7）による信頼感・清潔感
- 角丸カード、十分な余白
- 落ち着いたトーンのUI（怪しさの排除）

## 強いところ

- **YMYL対策UIが充実**: 免責バナー、確認日表示、リスクレベル表示、自動公開無効化の明示、詳細な運営ポリシー
- **料金比較テーブル**: 五十音順で並列比較。順位付け・推奨表現を一切排除
- **PR表記の徹底**: すべてのアフィリエイトリンクにPRラベルを付与
- **「おすすめ」「ランキング」を使わない設計**: 「公式情報比較」「料金比較」「条件比較」に統一
- **確認日一覧をサイドバーに常設**: 情報の鮮度が一目でわかる
- **怪しさを抑えたデザイン**: 攻撃的な赤を使わず、青・白・ミント・ベージュの落ち着いた配色

## 弱いところ

- **対策UIが多すぎてコンバージョンに影響する可能性**: 免責事項・注意書きが画面の多くを占め、アフィリエイトリンクへの誘導が弱くなる
- **美容系読者が硬い印象を持つリスク**: 信頼感を優先した結果、医療系サイトに近い印象になり、カジュアルな美容情報を求めるユーザーが離脱する可能性
- **コンテンツ制作コストが高い**: 確認日管理、リスクレベル付与、手動レビューなど運用負荷が大きい
- **差別化が「真面目さ」のみ**: 競合との差別化ポイントがコンプライアンスUIに偏り、コンテンツの面白さ・独自性が弱い

## 本番採用するなら

### 残す要素
- YMYL免責バナー（法的リスク回避に必須）
- 料金比較テーブル（五十音順・非ランキング）
- PR表記の徹底
- 確認日の明記
- 自動公開無効化の方針
- 青基調の配色

### 削る/調整する要素
- リスクレベル表示 → 全記事に付けるのは過剰。医療系（脱毛・AGA）のみに限定
- 運営ポリシーの分量 → 別ページに切り出し、サイドバーはリンクのみに
- 免責バナーの文量 → 1行に圧縮し、詳細はモーダルやツールチップで表示
- スレッド形式 → 美容ジャンルではビジュアル（Before/After等）の方が求められるため、画像対応を検討
- カラーにアクセント追加 → ミントやベージュだけでなく、美容系らしい柔らかいピンクやラベンダーを差し色に使うことを検討
