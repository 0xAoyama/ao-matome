# 004d — modernized 2ch matome

## 狙い
004（古典的2chまとめ）の構造・配色をほぼそのまま維持し、角丸・余白・pill型バッジ・行間だけを現代化。004と005の中間地点だが、ベースはあくまで004。「2chまとめを読み慣れた人が違和感なく使えるが、少しだけ今風」という着地。

## 強いところ
- 004の回遊性・情報密度・ランキング・サイドバー構造をすべて保持
- 角丸(6px)・余白増・pill型タグだけで印象が柔らかくなり、広告主ウケが改善する可能性
- 004からの移行コストが最小（CSS変数の追加程度）
- 2chまとめ読者の既視感を保ちつつ、新規ユーザーの「怪しさ」を軽減

## 弱いところ
- 変化が小さすぎて「これなら004でいい」という判断もありうる
- 角丸だけでは信頼感の本質的改善にはならない — コンテンツポリシーの方が重要
- 005寄りのモダンUIを期待するユーザーには中途半端に見える

## 本番採用するなら
- **残す**: 角丸・余白・pill型バッジ、2カラム構造、ランキング、サイドバー広告枠
- **削る**: 004dと004の差が小さいため、004自体をこの方向に更新する形で統合が合理的
- **追加検討**: CSS変数化（角丸・余白・配色をトークンで管理）、ダークモード対応
